Aide sur les paquets installés par l'utilisateur

Cette application est destinée à faciliter la réinstallation des paquets que l'utilisateur a ajoutés à l'installation par défaut.

Elle permet d'afficher la liste des paquets installés manuellement par l'utilisateur. Cette liste peut être sauvegardée dans un simple fichier texte.

De plus, l'application permet de charger une liste de paquets sauvegardée. La liste de paquets chargée est affichée pour sélection. L'utilisateur peut alors sélectionner les paquets à installer.

Surtout après une nouvelle installation du système, il est facile de sélectionner les paquets souhaités pour l'installation à partir d'une liste existante de paquets installés par l'utilisateur.

Il y a deux options à choisir :

  1. créer rapidement et facilement une liste de ces paquets
  2. utiliser cette liste à un autre endroit pour revoir et réinstaller ces paquets, s'ils sont encore disponibles

Créer une liste de paquets installés par l'utilisateur

  • Une liste des paquets installés par l'utilisateur est affichée.
  • Note : Packages of installed Linux kernels or NVIDIA graphics driver will not be listed.

  • Press [Save List] and select the location to save the package list.

Ouvrir une liste précédemment sauvegardée pour installer les paquets sélectionnés

  • Sélectionnez la liste de paquets que vous souhaitez installer.
  • La liste des logiciels s'affiche.
  • Décochez les paquets de la liste que vous ne souhaitez pas installer.
  • Une notification est fournie pour tout paquet manquant.

  • Press [OK] and the selected packages will be installed if still available.

Les paquets non disponibles seront listés

  • Les paquets non disponibles ou ne pouvant être installés seront listés.
  • Press [Save List] and select location to save the list of missing packages.

Paquets disponibles à installer

  • Le processus d'installation est lancé dans une nouvelle fenêtre de terminal.
  • L'authentification est requise pour effectuer l'installation.
  • La confirmation de l'installation est demandée. Appuyez sur [N] pour annuler l'installation.

  • Pour fermer la fenêtre du terminal après la fin du processus d'installation, appuyez sur [Enter].
