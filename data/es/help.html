Ayuda de Paquetes Instalados por el Usuario

Esta aplicacion esta pensada para facilitar la reinstalacion de paquetes que el usuario ha anadido a la instalacion por defecto.

Puede mostrar una lista de paquetes instalados manualmente por el usuario. Esta lista puede ser guardada en un simple archivo de texto.

Ademas, la aplicacion permite cargar una lista de paquetes guardada. La lista de paquetes cargada se muestra para su seleccion. El usuario puede ahora seleccionar que paquetes instalar.

Especialmente despues de una nueva instalacion del sistema, es facil seleccionar los paquetes deseados para su instalacion desde una lista existente de paquetes instalados por el usuario.

Hay dos opciones para elegir:

  1. crear rapida y facilmente una lista de esos paquetes
  2. usar esa lista en otro lugar para revisar y reinstalar esos paquetes, si todavia estan disponibles

Crear una lista de paquetes instalados por el usuario

  • Se muestra una lista de los paquetes instalados por el usuario.
  • Nota: Los paquetes de kernels Linux instalados o el driver grafico NVIDIA no seran listados.

  • Pulse [Save List] y seleccione la ubicacion para guardar la lista de paquetes.

Abrir una lista previamente guardada para instalar los paquetes seleccionados

  • Seleccione la lista de paquetes que desea instalar.
  • La lista de paquetes sera mostrada.
  • Desmarque los paquetes de la lista que no desea instalar.
  • Se provee una notificacion para cualquier paquete faltante.

  • Presione [OK] y los paquetes seleccionados seran instalados si todavia estan disponibles.

Los paquetes no disponibles seran listados

  • Los paquetes no disponibles o que no se pueden instalar seran listados.
  • Presione [Save List] y seleccione la ubicacion para guardar la lista de paquetes faltantes.

Paquetes disponibles con ser instalados

  • El proceso de instalacion se inicia dentro de una nueva ventana de terminal.
  • Se requiere autenticacion para realizar la instalacion.
  • Se requiere confirmacion para realizar la instalacion. Presione [N] para cancelar la instalacion.

  • Para cerrar la ventana de la terminal despues de que el proceso de instalacion termine presione [Enter].
