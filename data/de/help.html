Hilfe für benutzerinstallierte Pakete

Diese Anwendung ist dazu gedacht, die Neuinstallation von Paketen zu erleichtern, die der Benutzer zur Standardinstallation hinzugefügt hat.

Sie kann eine Liste der vom Benutzer manuell installierten Pakete anzeigen. Diese Liste kann in einer einfachen Textdatei gespeichert werden.

Außerdem erlaubt die Anwendung das Laden einer gespeicherten Paketliste. Die geladene Paketliste wird zur Auswahl angezeigt. Der Benutzer kann nun auswählen, welche Pakete er installieren möchte.

Besonders nach einer Neuinstallation des Systems ist es einfach, aus einer bestehenden Liste der vom Benutzer installierten Pakete die gewünschten Pakete zur Installation auszuwählen.

Es stehen zwei Optionen zur Auswahl:

  1. schnell und einfach eine Liste dieser Pakete erstellen
  2. diese Liste an einem anderen Ort zu verwenden, um diese Pakete zu überprüfen und neu zu installieren, falls noch verfügbar

Eine Liste der vom Benutzer installierten Pakete erstellen

  • Eine Liste der vom Benutzer installierten Pakete wird angezeigt.
  • Hinweis: Pakete von installierten Linux-Kerneln oder NVIDIA-Grafiktreibern werden nicht aufgelistet.

  • Drücken Sie [Liste speichern] und wählen Sie den Speicherort für die Paketliste.

Öffne eine zuvor gespeicherte Liste, um ausgewählte Pakete zu installieren

  • Wählen Sie die Paketliste aus, die Sie installieren möchten.
  • Die Paketliste wird angezeigt.
  • Deaktivieren Sie alle Pakete in der Liste, die Sie nicht installieren möchten.
  • Es wird eine Benachrichtigung über fehlende Pakete angezeigt.

  • Drücken Sie [OK] und die ausgewählten Pakete werden installiert, sofern noch verfügbar.

Nicht verfügbare Pakete werden aufgelistet

  • Nicht installierbare oder nicht installierbare Pakete werden aufgelistet.
  • Drücken Sie [Liste speichern] und wählen Sie den Speicherort für die Liste der fehlenden Pakete.

Verfügbare Pakete, die installiert werden können

  • Der Installationsprozess wird in einem neuen Terminalfenster gestartet.
  • Eine Authentifizierung zur Durchführung der Installation ist erforderlich.
  • Es wird eine Bestätigung für die Durchführung der Installation verlangt. Drücken Sie [N], um die Installation abzubrechen.

  • Um das Terminalfenster nach Abschluss der Installation zu schließen, drücken Sie [Enter].
